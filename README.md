# medniybik

Статическая одностраничная шуточная веб-страница ("пранк"). Пользователь видит экран с кнопками вида "Скачать RTX 1489 / RTX 1337 / RX 228"; по клику запускается анимация фейкового прогресса загрузки (растёт до 1489%), меняется фон и начинает играть аудиотрек.

Никакой реальной загрузки, сбора данных или сетевых запросов страница не выполняет — весь эффект чисто визуальный и клиентский.

## Технологии

Чистые HTML, CSS и JavaScript в одном файле `index.html`, без сборки и без зависимостей.

## Файлы

```
index.html    — вся разметка, стили и логика страницы
background.gif — фон, который включается после нажатия кнопки
aero-chord-surface_r3JBf9Yx.mp3 — аудиотрек, запускается и зацикливается после клика
```

## Запуск

Просто открыть `index.html` в браузере — сервер не требуется.

Страницу также можно захостить как статику (например, через GitHub Pages) — она не зависит от бэкенда.
